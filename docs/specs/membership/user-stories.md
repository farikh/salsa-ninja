# Membership: User Stories

## Registration Stories

### Must Have

**US-REG-01: QR Code Registration**
> As a potential member at the studio, I want to scan a QR code and quickly create an account so that I can join the community without complicated forms.

**US-REG-02: Referral Link Registration**
> As a person referred by an existing member, I want to click a referral link and create an account so that my friend gets credit for referring me.

**US-REG-03: Email Verification**
> As a new user, I want to verify my email via magic link so that I don't need to remember another password.

**US-REG-04: Profile Creation**
> As a new member, I want to set up my profile (name, photo, dance experience) so that other members can identify me.

### Should Have

**US-REG-05: Subscription Selection**
> As a new member, I want to choose my subscription plan during signup so that I can immediately access full features.

**US-REG-06: Onboarding Tour**
> As a new member, I want a quick tour of the platform so that I know where to find things.

### Could Have

**US-REG-07: Social Login**
> As a potential member, I want to sign up with Google so that signup is even faster.

## Authentication Stories

### Must Have

**US-AUTH-01: Magic Link Login**
> As a returning member, I want to enter my email and receive a login link so that I can access my account without remembering a password.

**US-AUTH-02: Session Persistence**
> As a logged-in member, I want to stay logged in for 7 days so that I don't have to log in every time I visit.

**US-AUTH-03: Logout**
> As a logged-in member, I want to log out so that I can secure my account on shared devices.

### Should Have

**US-AUTH-04: Remember Device**
> As a member, I want the platform to remember my device so that login is faster on my phone.

## Role & Access Stories

### Must Have

**US-ROLE-01: Role-Based UI**
> As a member, I want to only see features I have access to so that the interface isn't confusing with locked content.

**US-ROLE-02: Automatic Role Assignment**
> As a member, I want my role to automatically update when my subscription changes so that I don't have to contact the owner.

**US-ROLE-03: Permission Enforcement**
> As a member, I want the system to prevent me from accessing content I shouldn't see so that I know what's included in my subscription.

### Should Have

**US-ROLE-04: Upgrade Prompts**
> As a limited member, I want to see what I'm missing and how to upgrade so that I understand the value of a full subscription.

## Profile Stories

### Must Have

**US-PROF-01: View Own Profile** | **US-PROF-02: Edit Profile** | **US-PROF-03: View Other Profiles**

### Should Have

**US-PROF-04: Privacy Controls** | **US-PROF-05: Language Preference** (English/Spanish)

## Owner Stories

### Must Have

**US-OWN-01: View All Members** | **US-OWN-02: Member Details** | **US-OWN-03: Manual Role Assignment** | **US-OWN-04: Generate Invite Codes**

### Should Have

**US-OWN-05: Member Search** | **US-OWN-06: Bulk Actions** (tag multiple members)
