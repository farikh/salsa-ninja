# UI/UX Design

## Component Library

Using **shadcn/ui** components:
- Button, Card, Dialog, DropdownMenu
- Form, Input, Textarea, Select
- Tabs, Avatar, Badge, Calendar
- Toast, Sheet, Skeleton

## Color Palette

```css
/* Primary - Vibrant Red/Coral */
--primary-500: #ef4444;
--primary-600: #dc2626;

/* Secondary - Gold/Amber */
--secondary-500: #f59e0b;

/* Neutral */
--gray-50: #fafafa;
--gray-900: #18181b;
```

## Responsive Breakpoints

```css
sm: 640px   /* Large phones */
md: 768px   /* Tablets */
lg: 1024px  /* Laptops */
xl: 1280px  /* Desktops */
```
