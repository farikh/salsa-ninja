# Feature Spec: Admin Panel

Access: owner (full), instructor (partial)

## Sections

- **Dashboard:** Quick stats, recent activity
- **Event management:** Create, edit, recurring events
- **Video management:** Upload, tag, organize videos
- **Member management:** Roles, tags, subscription status
- **Announcements:** Create and target by member tags
- **Invite codes / QR generation:** Track marketing channels
- **Revenue dashboard:** (future phase)
